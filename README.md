# AI Agent Scraper & API

## Description
Ce projet est une application Python complète qui :
- Scrape des offres d’emploi depuis un site web.
- Nettoie et transforme les données collectées.
- Stocke les données dans une base PostgreSQL.
- Expose une API FastAPI pour interroger les données et générer des requêtes SQL avec un modèle d’IA (Mistral).

## Technologies
- Python 3.11
- FastAPI
- PostgreSQL
- Docker & Docker Compose
- Mistral AI

## Prérequis
- Docker et Docker Compose installés sur votre machine.
- Clé API Mistral AI.
